# User Management API Setup Tasks

- [x] Initialize npm project with package.json
- [x] Install required packages: express, mongoose, dotenv, bcryptjs, basic-auth
- [x] Install nodemon as dev dependency
- [x] Create .env file with PORT and MONGO_URI
- [x] Create config/db.js for MongoDB connection
- [x] Create models/User.js for user schema
- [x] Create middleware/auth.js for basic authentication
- [x] Create controllers/userController.js for CRUD operations
- [x] Create routes/userRoutes.js for API routes
- [x] Create server.js as the main entry point
- [x] Start the server with npx nodemon server.js
